# Annonces Facebook / Instagram — Sites web & E-commerce (PrimeDev Studios)

Tout pointe vers la landing **`/web/`**. Campagne **séparée** de celle de l'IA
(audience et message différents). Ton **humain, québécois, honnête**. On mène par la
douleur (site faible / inexistant = clients perdus), pas par le jargon technique.

> ⚠️ **Honnêteté** : pas de faux témoignages. Ton différenciateur = sur mesure (vs
> templates), « tu parles au développeur », et le **Programme Vitrine (2 places)** —
> une vraie offre à rabais, pas une fausse rareté.

---

## 1. Structure de campagne

| Niveau | Réglage |
|---|---|
| **Objectif** | **Prospects (Leads)** → **Site web** → optimiser l'événement **Lead** (Pixel branché). Au début, si peu de volume : optimiser **Vues de page de destination**, puis basculer. |
| **Budget** | **20-25 $/jour** (CBO). Laisser tourner **7 jours** avant de juger. |
| **Ensembles** | 1 pour démarrer. |
| **Créatifs** | 3-4 annonces (angles ci-dessous). |
| **Placements** | **Advantage+** (automatiques). |

> 💡 Le projet web vaut plus cher qu'un lead IA (4 500-12 000 $). Un coût par lead plus
> élevé (40-80 $) reste rentable : **un seul contrat couvre des mois de pub.**

## 4. Direction créative (visuel / vidéo)

- **Avant / après** d'un site (vieux/laid → moderne) : le format le plus parlant pour
  le web. Montre une vraie transformation.
- **Vidéo selfie 15-30 s** : toi qui montres un site sur ton écran/téléphone en
  scrollant — « Regarde la différence entre ça… et ça ». Authentique > léché.
- **Capture d'écran** d'un beau site responsive (desktop + mobile côte à côte).
- **Pour l'e-commerce** : montre le parcours d'achat (produit → panier → paiement).
- **Évite** les images IA génériques de « code » ou de robots : on vend du concret.

**Format** : 4:5 (Feed) + 9:16 (Reels/Stories). Sous-titres TOUJOURS.

---

## 5. Lancement & optimisation

1. **Avant de dépenser** : Pixel Helper sur `/web/`, accepte le consentement, remplis
   le formulaire test → vérifie **PageView** + **Lead**. Active le **Matching avancé
   automatique** dans le Gestionnaire d'événements.
2. **Semaine 1** : 20-25 $/jour, 4-5 annonces, ne touche à rien (phase d'apprentissage).
3. **Après 7 jours** : coupe les pires, garde les gagnantes, ajoute des angles.
4. **Rafraîchis** le créatif toutes les 2-4 semaines.
5. **Repère honnête** : un lead web à **40-80 $** est sain (le projet vaut des milliers
   de $). Un « lead » = une demande de soumission ; tu convertis à l'appel/au devis.

## 6. Deux campagnes, deux audiences

| | Campagne IA (`/ia/`) | Campagne Web (`/web/`) |
|---|---|---|
| Offre | Agent IA téléphone + chatbot | Sites & boutiques en ligne |
| Douleur | Appels manqués = leads perdus | Site faible/absent = clients perdus |
| Lead vaut | ~1 500 $/mois récurrent | 4 500-12 000 $ une fois |
| Coût/lead sain | 15-40 $ | 40-80 $ |

> Garde-les **séparées** dans Ads Manager : audiences et messages différents, sinon
> Meta optimise mal. Même Pixel, même page Facebook — juste 2 campagnes distinctes.
