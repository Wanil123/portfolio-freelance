# Portfolio Freelance — Wanil Parfait

Projet React + Vite + Tailwind + lucide-react.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Déploiement

- Déployer sur Netlify (build: `npm run build`, publish: `dist`)
- Modifier les images dans `public/assets/projects`
